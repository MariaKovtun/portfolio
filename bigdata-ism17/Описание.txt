Приложение для Apache Spark в автономном режиме (работая на одной машине без полной среды Hadoop) по анализу литературного текста:

1. Очистка текста

2. WordCount по тексту;

3. Top50 самых и наименее распространенных слов;

4. Stemming для текста (выделение основ c использованием библиотеки snowball);

5. Top50 самых и наименее распространенных слов по Stemming.
